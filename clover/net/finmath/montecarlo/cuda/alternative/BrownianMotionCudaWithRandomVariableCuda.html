26    /**

28    * <i>W = (W<sub>1</sub>,...,W<sub>n</sub>)</i> where <i>W<sub>i</sub></i> is
29    * a Brownian motion and <i>W<sub>i</sub></i>, <i>W<sub>j</sub></i> are
30    * independent for <i>i</i> not equal <i>j</i>.
31    *
32    * For a correlated Brownian motion with see
33    * {@link net.finmath.montecarlo.CorrelatedBrownianMotion}.
34    *
35    * Here the dimension <i>n</i> is called factors since this Brownian motion is used to
36    * generate multi-dimensional multi-factor Ito processes and there one might
37    * use a different number of factors to generate Ito processes of different
38    * dimension.
39    *
40    * The quadruppel (time discretization, number of factors, number of paths, seed)
41    * defines the state of an object of this class, i.e., BrownianMotionLazyInit for which
42    * there parameters agree, generate the same random numbers.
43    *
44    * The class is immutable and thread safe. It uses lazy initialization.
