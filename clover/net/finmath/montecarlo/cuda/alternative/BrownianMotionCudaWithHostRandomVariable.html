31    /**
32    * Implementation of a time-discrete n-dimensional Brownian motion
33    * <i>W = (W<sub>1</sub>,...,W<sub>n</sub>)</i> where <i>W<sub>i</sub></i> is
34    * a Brownian motion and <i>W<sub>i</sub></i>, <i>W<sub>j</sub></i> are
35    * independent for <i>i</i> not equal <i>j</i>.
36    *
37    * For a correlated Brownian motion with see
38    * {@link net.finmath.montecarlo.CorrelatedBrownianMotion}.
39    *
40    * Here the dimension <i>n</i> is called factors since this Brownian motion is used to
41    * generate multi-dimensional multi-factor Ito processes and there one might
42    * use a different number of factors to generate Ito processes of different
43    * dimension.
44    *
45    * The quadruppel (time discretization, number of factors, number of paths, seed)
46    * defines the state of an object of this class, i.e., BrownianMotionLazyInit for which
47    * there parameters agree, generate the same random numbers.
48    *
49    * The class is immutable and thread safe. It uses lazy initialization.
50    *
51    * @author Christian Fries
52    * @version 1.6
53    */
